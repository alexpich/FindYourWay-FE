User Stories
-    As a User, I want to be able to register and login, so that I may create and view my account
-    As a User, I want to be able to obtain points for a reward system, so that I may collect points and obtain badges/coupons
-    As a User, I want to be able to answer the website’s questions, so that I may receive a recommendation from the website
-    As a User, I want to be able to view a Help and about screen, so that I may receive assistance or learn more about the project
-    As a User, I want to be able to “favorite” a recent location that I have been to, so that I may have a reference for the future
-    As a User, I want to be able to change my personal info, so that I may update my profile
-    As a User, I want to be able to inspect establishments that are recommended, so that I may get more info on the establishment
-    As a User, I want to be able to view recommended establishments on a map, so that I may get a better idea of the establishment’s location
-    As a User, I want to be able to delete my own account, so that I may remove my account from the website.
-    As a Admin, I want to be able to view all Users, so that I may remove a User if needed
